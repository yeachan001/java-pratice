# 학점 관리 프로그램 개발

- 사용자로부터 점수를 입력 받아 학점을 주는 프로그램을 개발하시오.
	- 학점 부여 기준:
	    - 90점 이상: A 학점
	    - 80점 이상 90점 미만: B 학점
	    - 70점 이상 80점 미만: C 학점
	    - 60점 이상 70점 미만: D 학점
	    - 60점 미만: F 학점
	- 시스템은 결정된 학점을 "학점: X" 형식으로 출력해야 합니다.
- 함께 Figma 순서도 작성
- 손코드 필사 실시
